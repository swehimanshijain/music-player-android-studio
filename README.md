# Android Music Player App

A simple Android Music Player application built using Java. The app allows users to browse and play local audio files and manage their favorite songs.

## Features

- Play local music using MediaPlayer
- Display songs using RecyclerView
- Add and remove favorite songs
- Simple and user-friendly interface

## Technologies Used

- Java
- Android Studio
- Android SDK
- MediaPlayer
- RecyclerView

## Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle.
4. Run the app on an emulator or Android device.
